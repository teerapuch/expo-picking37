# expo-picking37
expo-picking37 วิธีสร้าง App Share รูปภาพ เป็นวิธีการทำงาน ด้วย React Native ที่เขียนขึ้นมาร่วมกับระบบ Expo ครอบเอาไว้ ทำให้สามารถทำงาน ได้อย่างง่ายดาย รวดเร็ว และสะดวกมากๆ
